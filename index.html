Пошаговая инструкция по настройке и запуску онлайн-игры

Настройка Firebase

1 Создание проекта Firebase

Перейдите на Firebase Console и нажмите "Добавить проект".

  • Введите название проекта (например, "Mafia Game")
  • Примите условия использования
  • Отключите Google Analytics (для простоты)
  • Нажмите "Создать проект"
2 Добавление веб-приложения

После создания проекта:

  1. Нажмите на значок </> (для веб-приложения)
  2. Введите название приложения (например, "Mafia Game")
  3. Нажмите "Зарегистрировать приложение"
  4. Скопируйте конфигурационный объект
const firebaseConfig = { apiKey: "AIzaSyDrOAhG2hQphTC8xmI_L1HD835Z101GPAI", authDomain: "grean-mafia.firebaseapp.com", databaseURL: "https://grean-mafia-default-rtdb.europe-west1.firebasedatabase.app", projectId: "grean-mafia", storageBucket: "grean-mafia.firebasestorage.app", messagingSenderId: "603308315781", appId: "1:603308315781:web:053d9551ed6516477905ec" };
3 Настройка базы данных

В меню слева выберите "Realtime Database":

  • Нажмите "Создать базу данных"
  • Выберите "Начните в тестовом режиме"
  • Нажмите "Готово"
Совет: В тестовом режиме база данных открыта для чтения/записи. Для продакшена настройте правила безопасности.
4 Настройка правил безопасности

На вкладке "Правила" замените содержимое на:

{
  "rules": {
    "rooms": {
      ".read": true,
      ".write": true
    }
  }
}

Нажмите "Опубликовать".

Внимание: Эти правила разрешают всем читать и писать в базу данных. Для продакшена настройте более строгие правила.

Запуск игры

5 Вставка конфигурации в игру

В коде игры найдите следующий блок и замените его на ваш конфиг:

// Замените этот конфиг на ваш из Firebase Console
const firebaseConfig = {
  apiKey: "ВАШ_API_KEY",
  authDomain: "ВАШ-ПРОЕКТ.firebaseapp.com",
  databaseURL: "https://ВАШ-ПРОЕКТ-default-rtdb.firebaseio.com",
  projectId: "ВАШ-ПРОЕКТ",
  storageBucket: "ВАШ-ПРОЕКТ.appspot.com",
  messagingSenderId: "ВАШ_SENDER_ID",
  appId: "ВАШ_APP_ID"
};
6 Размещение игры

Вы можете разместить игру на:

  • Firebase Hosting (рекомендуется)
  • GitHub Pages
  • Любом другом статическом хостинге

Для Firebase Hosting:

# Установите Firebase CLI
npm install -g firebase-tools

# Войдите в Firebase
firebase login

# Инициализируйте проект
firebase init

# Выберите Hosting
# Укажите папку с файлами (public)
# Разместите приложение
firebase deploy

Для GitHub Pages:

  1. Создайте репозиторий на GitHub
  2. Загрузите файл index.html
  3. В настройках репозитория перейдите в "Pages"
  4. Выберите ветку и папку с файлом

Как играть

Основные шаги:

1. Создание игры

Нажмите "Создать новую игру", скопируйте ID комнаты

2. Приглашение

Отправьте ID комнаты друзьям

3. Начало игры

Когда все присоединятся, нажмите "Начать игру"

Игровой процесс

Ночная фаза

Мафия выбирает жертву, шериф проверяет игроков

Дневная фаза

Игроки обсуждают и голосуют за подозреваемого

Конец игры

Мирные жители или мафия побеждают

Структура кода

// Основные компоненты игры:

firebaseConfig - конфигурация Firebase
gameState - текущее состояние игры
initGame() - инициализация игры
createRoom() - создание игровой комнаты
joinRoom() - присоединение к комнате
startGame() - начало игры
nextPlayer() - переход к следующему игроку
endGame() - завершение игры

// Firebase функции:
saveGameState() - сохранение состояния в БД
loadGameState() - загрузка состояния из БД
listenForGameChanges() - отслеживание изменений

Важные замечания

Безопасность: В тестовом режиме Firebase база данных открыта для записи. Для продакшена обязательно настройте правила аутентификации.
Производительность: Игра рассчитана на 5 игроков. При большем количестве может потребоваться оптимизация кода.
